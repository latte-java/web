# OpenID Connect Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Status (2026-04-27):** This plan was the original intent. The actual implementation diverged in several places — see the **"Implementation status & deltas"** section immediately below for the authoritative shape of the shipped code. The phase-by-phase task list further down is preserved for historical context but should not be treated as a guide to current code structure or coverage.

**Goal:** Implement OIDC authentication for Latte Web — config + discovery + login-redirect + callback + token validation (two modes) + refresh + role-based authorization + logout — all IdP-agnostic with no external OIDC/OAuth2 library. Tested against a local FusionAuth instance (provisioned by a kickstart at `src/test/docker/kickstart/kickstart.json`) with a few Mock-IdP tests for failure modes not reproducible against real FusionAuth.

**Spec:** [docs/design/oidc.md](../../design/oidc.md).

### API & semantic deltas vs. this plan

- **Class renamed.** `OpenIDConnect<U>` → `OIDC<U>`. Static accessors: `OIDC.jwt()`, `OIDC.optionalJWT()`. Instance accessors: `oidc.user()`, `oidc.optionalUser()`.
- **`ScopedValue<JWT>` lives on `Tools`**, not on `OIDC`. `OIDC` exposes it through the static and instance accessors above.
- **Protection middlewares have package-private constructors.** `Authenticated`, `HasAnyRole`, `HasAllRoles` cannot be instantiated outside the `org.lattejava.web.oidc` package. Construction goes through `oidc.authenticated()`, `oidc.hasAnyRole(...)`, `oidc.hasAllRoles(...)`.
- **`OIDCConfig` field changes:**
  - Added `loginPath` (default `/login`) — see the new login-redirect routing below.
  - Removed the originally-planned `redirectURI` field. The redirect URI is always computed per-request as `req.getBaseURL() + callbackPath` via `OIDCConfig.fullRedirectURI(req)`.
  - `roleExtractor` type is `Function<JWT, Set<String>>` (was `Function<JWT, Iterable<String>>`). Default: `jwt -> new HashSet<>(jwt.getList("roles", String.class))`.
- **Localhost HTTP allowed.** `Tools.requireSecureURI` permits `http://` only when the host is `localhost` / `127.0.0.1` / `::1`. Non-loopback HTTP is rejected with `IllegalArgumentException`.
- **Discovery runs at `OIDCConfig.Builder.build()`**, not at `OIDC.create(...)`. Failures wrap into `IllegalStateException("Failed to fetch OIDC discovery document for issuer [...]")`. After discovery, `build()` requires `authorize`/`token`/`jwks` to be resolved; `userinfo` is required only when `validateAccessToken=false`.
- **Path validation.** Builder checks that `callbackPath`, `logoutPath`, and `logoutReturnPath` start with `/` and are pairwise distinct. `loginPath` is not checked for `/` prefix or distinctness.
- **Login redirect now goes through `loginPath`.** `Authenticated`, on a missing access token (or after refresh failure), sets the `oidc_return_to` cookie and `res.sendRedirect(config.loginPath())`. The system middleware (`OIDC.handle`) then dispatches to `LoginHandler`, which builds the authorize URL with PKCE and redirects out. This is a structural change from the original "Authenticated builds the authorize URL itself" plan: the return-to cookie is set by `Authenticated`, not by the login redirect path.
- **Cookie attributes.** `Path=/`, `SameSite=Strict`. `Secure` is conditional on `req.getScheme() == "https"` or `X-Forwarded-Proto: https` (was specified as unconditional in the original plan). State and return-to cookies are set as transient (no Max-Age) HttpOnly. `id_token` is non-HttpOnly per plan; `access_token` and `refresh_token` are HttpOnly per plan. Max-Age values per plan: `expires_in` for access/id, `refreshTokenMaxAge` for refresh.
- **Role-check failures return 401, not 403.** `HasAnyRole` / `HasAllRoles` return `401` both when no JWT is bound (configuration error: `Authenticated` wasn't installed upstream) and when the role set doesn't satisfy the predicate. The original plan called for `403` on the latter.
- **Callback `validateAccessToken=true` re-decodes the id_token, not the access token.** `CallbackHandler.handle` calls `new JWTDecoder().decode(idToken, jwks)` a second time when `validateAccessToken=true`, rather than decoding the access token. Cookies are still set; the runtime path (`Authenticated` + `TokenValidator`) does validate the access token on every protected request.
- **JWKS rotation handling is not implemented in the OIDC layer.** JWKS is fetched once at `OIDC.create(...)` and never refreshed. The original plan called for hourly background refresh + refresh-on-miss; neither is wired up in the shipped code.
- **`TokenValidator` returns a sealed `Result`.** `Valid(JWT) | Invalid | NetworkError`. Local mode performs JWT decode against JWKS plus an audience check (`jwt.audience().contains(config.clientId())`). Userinfo mode does `200 → wrap+audience check`, `401 → Invalid`, anything else → `NetworkError`. `iss`/`exp`/`nbf` checks come from `JWTDecoder` and the JWKS-backed verifier; the application layer adds only the audience check.
- **Refresh flow.** Lives in `Authenticated.attemptRefresh`. On any failure path (no refresh-token cookie, network exception, non-2xx response, unparseable body, missing new access_token, re-validation fails) the middleware clears auth cookies, sets the return-to cookie, and redirects to `loginPath`. Successful refresh calls `Tools.addAuthCookies(...)` with whatever subset of new tokens came back (null arguments are skipped, preserving existing cookies for omitted fields). Refresh is reactive only.
- **Cookie helpers consolidated in `Tools`.** `addAuthCookies` (id/access/refresh), `addTransientCookie` (state/return-to), `clearCookie` (single), `clearAllAuthCookies` (id/access/refresh), `clearAllCookies` (auth + state + return-to).
- **System middleware paths.** `OIDC.handle` dispatches to `callbackPath`, `loginPath`, `logoutPath`, `logoutReturnPath` (in that order in the source) and falls through to `chain.next` on any other path.
- **Logout endpoint discovery.** Discovery's `end_session_endpoint`, when present, populates `logoutEndpoint` automatically.

### Behavior the original plan called out and the code does correctly

- `Builder.build()` validation: blank/null `clientId`/`clientSecret`, missing-issuer-and-endpoints, missing `openid` scope, null `roleExtractor`, blank/duplicate cookie names, malformed/duplicate paths, non-HTTPS URIs (with the localhost carve-out described above), `validateAccessToken=false` without `userinfoEndpoint`.
- `UnauthenticatedException` lives in `org.lattejava.web` and is mapped to `401` by `ExceptionHandler` by default; user-supplied entries override on key collision; lookup walks the class hierarchy.
- State + return-to cookies as a transient single-value PKCE verifier; 22 random bytes → 44 hex chars; `code_challenge = base64url(sha256(state))` no padding.
- Callback: error-query handling, state mismatch → 400, missing code → 400, exception → 500, non-2xx → 400, missing `access_token`/`id_token` → 400, id-token signature verification → 400 on failure, return-to cookie honored.
- Logout: with `logoutEndpoint`, redirect to the IdP with `post_logout_redirect_uri`, `client_id`, optional `id_token_hint`. Without `logoutEndpoint`, clear cookies and redirect to `postLogoutLanding`. `LogoutReturnHandler` clears all cookies and redirects to `postLogoutLanding` regardless of how it was reached.

- Users:

  - **standard** — default settings, refresh-token rotation disabled

  - **short-ttl** — dedicated tenant with a 1-second access-token expiration for refresh-flow tests

public OIDCConfig build() {
  if (clientId == null || clientId.isBlank()) {
    throw new IllegalArgumentException("clientId must not be null or blank");
  }
  if (clientSecret == null || clientSecret.isBlank()) {
    throw new IllegalArgumentException("clientSecret must not be null or blank");
  }

  boolean hasIssuer = issuer != null && !issuer.isBlank();
  boolean hasAllEndpoints = authorizeEndpoint != null && tokenEndpoint != null
      && userinfoEndpoint != null && jwksEndpoint != null;
  if (!hasIssuer && !hasAllEndpoints) {
    throw new IllegalArgumentException(
        "Either [issuer] must be set or all four endpoints (authorize, token, userinfo, jwks) must be set");
  }

  if (scopes == null || !scopes.contains("openid")) {
    throw new IllegalArgumentException("scopes must contain [openid]");
  }
  if (roleExtractor == null) {
    throw new IllegalArgumentException("roleExtractor must not be null");
  }

  Set<String> cookieNames = new HashSet<>();
  for (String name : List.of(stateCookieName, accessTokenCookieName, refreshTokenCookieName,
      idTokenCookieName, returnToCookieName)) {
    if (name == null || name.isBlank()) {
      throw new IllegalArgumentException("cookie names must not be null or blank");
    }
    if (!cookieNames.add(name)) {
      throw new IllegalArgumentException("duplicate cookie name: [" + name + "]");
    }
  }

  Set<String> paths = new HashSet<>();
  for (String path : List.of(callbackPath, logoutPath, logoutReturnPath)) {
    if (path == null || !path.startsWith("/")) {
      throw new IllegalArgumentException("path must start with [/]: [" + path + "]");
    }
    if (!paths.add(path)) {
      throw new IllegalArgumentException("duplicate path: [" + path + "]");
    }
  }

  requireHTTPS("issuer", issuer == null ? null : URI.create(issuer));
  requireHTTPS("redirectURI", redirectURI);
  requireHTTPS("logoutEndpoint", logoutEndpoint);

  return new OIDCConfig(issuer, authorizeEndpoint, tokenEndpoint, userinfoEndpoint, jwksEndpoint,
      logoutEndpoint, clientId, clientSecret, redirectURI, scopes, roleExtractor,
      validateAccessToken, postLoginLanding, postLogoutLanding, callbackPath, logoutPath,
      logoutReturnPath, stateCookieName, accessTokenCookieName, refreshTokenCookieName,
      idTokenCookieName, returnToCookieName, refreshTokenMaxAge);
}

private static void requireHTTPS(String field, URI uri) {
  if (uri == null) {
    return;
  }
  if (!"https".equalsIgnoreCase(uri.getScheme())) {
    throw new IllegalArgumentException(field + " must use https: [" + uri + "]");
  }

package org.lattejava.web.oidc;

import module java.base;
import module java.net.http;

import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.ObjectMapper;

/**
 * Fetches and parses an OIDC Discovery document from {@code {issuer}/.well-known/openid-configuration}.
 */
class DiscoveryClient {
  private static final ObjectMapper MAPPER = new ObjectMapper();

  private final HttpClient http;

  DiscoveryClient(HttpClient http) {
    this.http = http;
  }

  /**
   * @throws IOException or InterruptedException on network failure; IllegalStateException on
   *     malformed document.
   */
  DiscoveryDocument fetch(String issuer) throws IOException, InterruptedException {
    String base = issuer.endsWith("/") ? issuer.substring(0, issuer.length() - 1) : issuer;
    URI wellKnown = URI.create(base + "/.well-known/openid-configuration");
    HttpRequest req = HttpRequest.newBuilder(wellKnown).GET().build();
    HttpResponse<String> res = http.send(req, HttpResponse.BodyHandlers.ofString());
    if (res.statusCode() != 200) {
      throw new IllegalStateException("Discovery fetch returned status [" + res.statusCode()
          + "] for [" + wellKnown + "]");
    }
    JsonNode node = MAPPER.readTree(res.body());
    return new DiscoveryDocument(
        uri(node, "authorization_endpoint"),
        uri(node, "token_endpoint"),
        uri(node, "userinfo_endpoint"),
        uri(node, "jwks_uri"),
        optionalUri(node, "end_session_endpoint"));
  }

  private static URI uri(JsonNode node, String field) {
    JsonNode v = node.get(field);
    if (v == null || v.isNull()) {
      throw new IllegalStateException("Discovery document missing required field [" + field + "]");
    }
    return URI.create(v.asText());
  }

  private static URI optionalUri(JsonNode node, String field) {
    JsonNode v = node.get(field);
    return (v == null || v.isNull()) ? null : URI.create(v.asText());
  }
}

/*
 * Copyright (c) 2025-2026, Latte Java, All Rights Reserved
 *
 * Licensed under the MIT License. See LICENSE for details.

package org.lattejava.web.oidc;

import module java.base;
import module java.net.http;

import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.ObjectMapper;

import java.math.BigInteger;
import java.security.KeyFactory;
import java.security.PublicKey;
import java.security.interfaces.RSAPublicKey;
import java.security.spec.RSAPublicKeySpec;
import java.util.Base64;

/**
 * Thread-safe cache of JWKS public keys keyed by {@code kid}. Background refresh runs hourly on a
 * daemon thread; an explicit {@link #refresh()} is called on unknown-kid lookups by the middleware.
 */
class JWKSCache {
  private static final ObjectMapper MAPPER = new ObjectMapper();

  private final HttpClient http;
  private final URI jwksUri;
  private final AtomicReference<Map<String, PublicKey>> keys = new AtomicReference<>(Map.of());

  JWKSCache(HttpClient http, URI jwksUri) {
    this.http = http;
    this.jwksUri = jwksUri;
  }

  Optional<PublicKey> get(String kid) {
    return Optional.ofNullable(keys.get().get(kid));
  }

  Set<String> knownKeyIds() {
    return keys.get().keySet();
  }

  /**
   * Fetches the JWKS document and replaces the cache atomically. Safe to call concurrently; last
   * writer wins.
   *
   * @throws IOException or InterruptedException on fetch failure.
   * @throws IllegalStateException on malformed JWKS.
   */
  void refresh() throws IOException, InterruptedException {
    HttpRequest req = HttpRequest.newBuilder(jwksUri).GET().build();
    HttpResponse<String> res = http.send(req, HttpResponse.BodyHandlers.ofString());
    if (res.statusCode() != 200) {
      throw new IllegalStateException("JWKS fetch returned status [" + res.statusCode()
          + "] for [" + jwksUri + "]");
    }
    JsonNode doc = MAPPER.readTree(res.body());
    JsonNode arr = doc.get("keys");
    if (arr == null || !arr.isArray()) {
      throw new IllegalStateException("JWKS document missing [keys] array");
    }
    Map<String, PublicKey> next = new HashMap<>();
    for (JsonNode key : arr) {
      String kid = textOrNull(key, "kid");
      String kty = textOrNull(key, "kty");
      if (kid == null || !"RSA".equals(kty)) {
        continue;  // non-RSA keys not supported in v1
      }
      String n = textOrNull(key, "n");
      String e = textOrNull(key, "e");
      if (n == null || e == null) {
        continue;
      }
      try {
        BigInteger modulus = new BigInteger(1, Base64.getUrlDecoder().decode(n));
        BigInteger exponent = new BigInteger(1, Base64.getUrlDecoder().decode(e));
        RSAPublicKey pub = (RSAPublicKey) KeyFactory.getInstance("RSA")
            .generatePublic(new RSAPublicKeySpec(modulus, exponent));
        next.put(kid, pub);
      } catch (Exception ex) {
        // Skip unparseable keys; other valid keys still load.
      }
    }
    keys.set(Map.copyOf(next));
  }

  private static String textOrNull(JsonNode node, String field) {
    JsonNode v = node.get(field);
    return (v == null || v.isNull()) ? null : v.asText();
  }
}

      if (az == null) az = doc.authorizeEndpoint();
      if (tk == null) tk = doc.tokenEndpoint();
      if (ui == null) ui = doc.userinfoEndpoint();
      if (jk == null) jk = doc.jwksEndpoint();
      if (lo == null) lo = doc.endSessionEndpoint();
    }

    if (az == null || tk == null || jk == null) {
      throw new IllegalStateException(
          "Required endpoint unresolved — set issuer or provide explicit authorize/token/jwks endpoints");
    }
    if (!config.validateAccessToken() && ui == null) {
      throw new IllegalStateException(
          "validateAccessToken=false requires userinfoEndpoint — set explicitly or provide issuer for discovery");
    }

    this.authorizeEndpoint = az;
    this.tokenEndpoint = tk;
    this.userinfoEndpoint = ui;
    this.jwksEndpoint = jk;
    this.logoutEndpoint = lo;

    this.jwks = new JWKSCache(http, jk);
    this.jwks.refresh();
  }

  public void handle(HTTPRequest req, HTTPResponse res, MiddlewareChain chain) throws Exception {
    String path = req.getPath();

    if (path.equals(config.logoutReturnPath())) { handleLogoutReturn(req, res); return; }

/**
 * Middleware that requires a valid authenticated session. Unauthenticated requests trigger an
 * OIDC authorize-redirect; authenticated requests bind the current JWT to the {@code OpenIDConnect}
 * ScopedValue and invoke the downstream chain.
 *
 * @author Brian Pontarelli
 */
public class Authenticated implements Middleware {
  private static final SecureRandom RANDOM = new SecureRandom();
  private static final HexFormat HEX = HexFormat.of();

  final OIDC<?> oidc;

  public Authenticated(OIDC<?> oidc) {
    this.oidc = Objects.requireNonNull(oidc, "oidc must not be null");
  }

  @Override
  public void handle(HTTPRequest req, HTTPResponse res, MiddlewareChain chain) throws Exception {
    redirectToLogin(req, res);  // Phase 7 replaces this with the full logic
  }

  /**
   * Starts a login flow: generates state+PKCE verifier, sets state + returnTo cookies, redirects
   * to the IdP's authorize endpoint.
   */
  void redirectToLogin(HTTPRequest req, HTTPResponse res) throws Exception {
    byte[] stateBytes = new byte[22];  // 22 bytes -> 44 hex chars -> 128 bits entropy, satisfies PKCE >=43 chars
    RANDOM.nextBytes(stateBytes);
    String state = HEX.formatHex(stateBytes);
    String codeChallenge = computeCodeChallenge(state);

    OIDCConfig config = oidc.config();
    URI redirectURI = config.redirectURI() != null
        ? config.redirectURI()
        : URI.create(req.getBaseURL() + config.callbackPath());

    // Cookies
    setTransientCookie(res, config.stateCookieName(), state, true);
    setTransientCookie(res, config.returnToCookieName(), req.getBaseURL() + req.getPath(), true);

    // Build authorize URL
    StringBuilder url = new StringBuilder(oidc.authorizeEndpoint().toString());
    url.append('?');
    url.append("response_type=code");
    url.append("&client_id=").append(URLEncoder.encode(config.clientId(), StandardCharsets.UTF_8));
    url.append("&redirect_uri=").append(URLEncoder.encode(redirectURI.toString(), StandardCharsets.UTF_8));
    url.append("&scope=").append(URLEncoder.encode(String.join(" ", config.scopes()), StandardCharsets.UTF_8));
    url.append("&state=").append(state);
    url.append("&code_challenge=").append(codeChallenge);
    url.append("&code_challenge_method=S256");

    res.sendRedirect(url.toString());
  }

  private static String computeCodeChallenge(String verifier) throws Exception {
    byte[] digest = MessageDigest.getInstance("SHA-256")
                                 .digest(verifier.getBytes(StandardCharsets.UTF_8));
    return Base64.getUrlEncoder().withoutPadding().encodeToString(digest);
  }

  static void setTransientCookie(HTTPResponse res, String name, String value, boolean httpOnly) {
    Cookie c = new Cookie(name, value);
    c.setPath("/");
    c.setSecure(true);
    c.setSameSite(SameSite.Strict);
    c.setHttpOnly(httpOnly);
    // No Max-Age — session cookie, browser discards on close; we also clear explicitly on use.
    res.addCookie(c);
  }

  static void clearCookie(HTTPResponse res, String name) {
    Cookie c = new Cookie(name, "");
    c.setPath("/");
    c.setSecure(true);
    c.setSameSite(SameSite.Strict);
    c.setMaxAge(0L);
    res.addCookie(c);
  }
}

import com.fasterxml.jackson.databind.ObjectMapper;

/**
 * Thin client around the OIDC token endpoint. Supports authorization-code exchange and refresh-token
 * grants. Client credentials sent via HTTP Basic auth (per RFC 6749 § 2.3.1).
 */
class TokenEndpointClient {
  private static final ObjectMapper MAPPER = new ObjectMapper();

  private final HttpClient http;
  private final URI tokenEndpoint;
  private final String basicAuth;

  TokenEndpointClient(HttpClient http, URI tokenEndpoint, String clientId, String clientSecret) {
    this.http = http;
    this.tokenEndpoint = tokenEndpoint;
    String creds = clientId + ":" + clientSecret;
    this.basicAuth = "Basic " + Base64.getEncoder()
        .encodeToString(creds.getBytes(StandardCharsets.UTF_8));
  }

  TokenResponse exchangeCode(String code, String redirectURI, String codeVerifier)
      throws IOException, InterruptedException {
    Map<String, String> form = Map.of(
        "grant_type", "authorization_code",
        "code", code,
        "redirect_uri", redirectURI,
        "code_verifier", codeVerifier);
    return post(form);
  }

  TokenResponse refresh(String refreshToken) throws IOException, InterruptedException {
    Map<String, String> form = Map.of(
        "grant_type", "refresh_token",
        "refresh_token", refreshToken);
    return post(form);
  }

  private TokenResponse post(Map<String, String> form) throws IOException, InterruptedException {
    String body = form.entrySet().stream()
        .map(e -> URLEncoder.encode(e.getKey(), StandardCharsets.UTF_8) + "="
            + URLEncoder.encode(e.getValue(), StandardCharsets.UTF_8))
        .reduce((a, b) -> a + "&" + b)
        .orElse("");
    HttpRequest req = HttpRequest.newBuilder(tokenEndpoint)
        .header("Authorization", basicAuth)
        .header("Content-Type", "application/x-www-form-urlencoded")
        .POST(HttpRequest.BodyPublishers.ofString(body))
        .build();
    HttpResponse<String> res = http.send(req, HttpResponse.BodyHandlers.ofString());
    return new TokenResponse(res.statusCode(), res.body());
  }

  record TokenResponse(int statusCode, String body) {
    boolean isSuccess() { return statusCode >= 200 && statusCode < 300; }

    JsonNode json() throws IOException { return MAPPER.readTree(body); }
  }
}

/**
 * Calls the OIDC userinfo endpoint with a bearer access token and returns a Result.

class UserinfoClient {
  private static final ObjectMapper MAPPER = new ObjectMapper();

  private final HttpClient http;
  private final URI userinfoEndpoint;

  UserinfoClient(HttpClient http, URI userinfoEndpoint) {
    this.http = http;
    this.userinfoEndpoint = userinfoEndpoint;
  }

  enum Status { VALID, INVALID, NETWORK_ERROR }

  record Result(Status status, Map<String, Object> claims) { }

  Result call(String accessToken) {
    try {
      HttpRequest req = HttpRequest.newBuilder(userinfoEndpoint)
          .header("Authorization", "Bearer " + accessToken)
          .GET()
          .build();
      HttpResponse<String> res = http.send(req, HttpResponse.BodyHandlers.ofString());
      int status = res.statusCode();
      if (status == 200) {
        JsonNode json = MAPPER.readTree(res.body());
        Map<String, Object> claims = MAPPER.convertValue(json,
            new com.fasterxml.jackson.core.type.TypeReference<Map<String, Object>>() {});
        return new Result(Status.VALID, claims);
      }
      if (status == 401) {
        return new Result(Status.INVALID, Map.of());
      }
      return new Result(Status.NETWORK_ERROR, Map.of());
    } catch (Exception e) {
      return new Result(Status.NETWORK_ERROR, Map.of());
    }
  }

private JWT attemptRefresh(HTTPRequest req, HTTPResponse res) {
  String refreshToken = readCookie(req, oidc.config().refreshTokenCookieName());
  if (refreshToken == null) return null;

  try {
    TokenEndpointClient client = new TokenEndpointClient(
        oidc.http(), oidc.tokenEndpoint(),
        oidc.config().clientId(), oidc.config().clientSecret());
    TokenEndpointClient.TokenResponse tok = client.refresh(refreshToken);
    if (!tok.isSuccess()) return null;

    com.fasterxml.jackson.databind.JsonNode body = tok.json();
    String newAccess = body.get("access_token").asText();
    long expiresIn = body.has("expires_in") ? body.get("expires_in").asLong() : 3600L;
    String newRefresh = body.has("refresh_token") ? body.get("refresh_token").asText() : null;
    String newIdToken = body.has("id_token") ? body.get("id_token").asText() : null;

    // Verify new access token the same way as a fresh one.
    JWT newAccessJWT;
    if (oidc.config().validateAccessToken()) {
      newAccessJWT = validateLocally(newAccess);
      if (newAccessJWT == null) return null;
    } else {
      try {
        newAccessJWT = validateByUserinfo(newAccess);
        if (newAccessJWT == null) return null;
      } catch (UserinfoUnavailableException e) {
        return null;
      }
    }

    setAuthCookie(res, oidc.config().accessTokenCookieName(), newAccess, expiresIn, true);
    if (newRefresh != null) {
      setAuthCookie(res, oidc.config().refreshTokenCookieName(), newRefresh,
          oidc.config().refreshTokenMaxAge().toSeconds(), true);
    }
    if (newIdToken != null) {
      setAuthCookie(res, oidc.config().idTokenCookieName(), newIdToken, expiresIn, false);
    }
    return newAccessJWT;
  } catch (Exception e) {
    return null;
  }
}

private static void setAuthCookie(HTTPResponse res, String name, String value,
    long maxAgeSeconds, boolean httpOnly) {
  Cookie c = new Cookie(name, value);
  c.setPath("/");
  c.setSecure(true);
  c.setSameSite(SameSite.Strict);
  c.setHttpOnly(httpOnly);
  c.setMaxAge(maxAgeSeconds);
  res.addCookie(c);
}

package org.lattejava.web.oidc;

import module java.base;
import module org.lattejava.http;

import org.lattejava.jwt.JWT;
import org.lattejava.web.*;

/**
 * Middleware requiring a valid session AND at least one of the given roles. Unauthenticated →
 * redirect to login; authenticated but missing role → 403.
 */
public class HasAnyRole implements Middleware {
  private final Authenticated authenticated;
  private final Set<String> required;

  public HasAnyRole(OIDC<?> oidc, String... roles) {
    if (roles == null || roles.length == 0) {
      throw new IllegalArgumentException("At least one role must be provided");
    }
    this.authenticated = new Authenticated(oidc);
    this.required = Set.of(roles);
  }

  @Override
  public void handle(HTTPRequest req, HTTPResponse res, MiddlewareChain chain) throws Exception {
    authenticated.handle(req, res, (req2, res2) -> {
      JWT jwt = OIDC.jwt();
      Set<String> have = rolesOf(authenticated.oidc, jwt);
      for (String r : required) {
        if (have.contains(r)) {
          chain.next(req2, res2);
          return;
        }
      }
      res2.setStatus(403);
    });
  }

  static Set<String> rolesOf(OIDC<?> oidc, JWT jwt) {
    Iterable<String> raw = oidc.config().roleExtractor().apply(jwt);
    if (raw == null) return Set.of();
    Set<String> out = new HashSet<>();
    for (String r : raw) out.add(r);
    return out;
  }
}

private void handleLogout(HTTPRequest req, HTTPResponse res) throws Exception {
  if (logoutEndpoint == null) {
    // Inline fall-through: clear cookies, redirect to postLogoutLanding.
    clearAllAuthCookies(res);
    res.sendRedirect(config.postLogoutLanding());
    return;
  }

  String idToken = readCookie(req, config.idTokenCookieName());
  URI returnURI = URI.create(req.getBaseURL() + config.logoutReturnPath());

  StringBuilder url = new StringBuilder(logoutEndpoint.toString());
  url.append('?');
  url.append("post_logout_redirect_uri=")
      .append(URLEncoder.encode(returnURI.toString(), StandardCharsets.UTF_8));
  url.append("&client_id=")
      .append(URLEncoder.encode(config.clientId(), StandardCharsets.UTF_8));
  if (idToken != null) {
    url.append("&id_token_hint=").append(URLEncoder.encode(idToken, StandardCharsets.UTF_8));
  }
  res.sendRedirect(url.toString());
}

private void handleLogoutReturn(HTTPRequest req, HTTPResponse res) throws Exception {
  clearAllAuthCookies(res);
  Authenticated.clearCookie(res, config.stateCookieName());
  Authenticated.clearCookie(res, config.returnToCookieName());
  res.sendRedirect(config.postLogoutLanding());
}

private void clearAllAuthCookies(HTTPResponse res) {
  Authenticated.clearCookie(res, config.accessTokenCookieName());
  Authenticated.clearCookie(res, config.idTokenCookieName());
  Authenticated.clearCookie(res, config.refreshTokenCookieName());
}
